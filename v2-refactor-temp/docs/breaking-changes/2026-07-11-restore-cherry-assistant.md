## What changed

Profiles with no prior agent or session history now receive the Cherry Assistant once automatically.
Profiles with existing history, or profiles where the assistant was deleted, are unchanged.

## Why this matters to the user

A new profile starts with Cherry Assistant available in the Agents area, without requiring manual setup.
Existing profiles and profiles that deleted Cherry Assistant do not receive it again.

## What the user should do

Nothing — automatic.
